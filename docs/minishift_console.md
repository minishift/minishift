## minishift console

Opens or displays the OpenShift Web Console URL.

### Synopsis


Opens the OpenShift Web Console URL in the default browser or displays it to the console.

```
minishift console
```

### Options

```
      --url   Prints the OpenShift Web Console URL to the console.
```
